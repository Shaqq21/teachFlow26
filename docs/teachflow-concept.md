# TeachFlow — Полная концепция SaaS-платформы для учителей

---

## 1. КРАТКОЕ РЕЗЮМЕ

**TeachFlow** — это веб-платформа для учителей, которая помогает экономить до 5 часов в неделю на подготовку уроков, обмениваться готовыми материалами с коллегами, управлять заданиями и поддерживать связь с учениками и родителями в одном месте.

**Целевая аудитория:**
- Учителя начальной школы (1–4 классы)
- Учителя средней и старшей школы (5–11 классы)
- Репетиторы и частные преподаватели
- Методисты и администрация школ

**Ключевое ценностное предложение:**
> «Готовые уроки за минуты, а не часы. Делитесь лучшими материалами и сосредоточьтесь на том, что важно — на учениках.»

## 3. MVP ФУНКЦИИ

| Приоритет | Функция | Описание |
|-----------|---------|----------|
| 🔴 High | **Библиотека шаблонов уроков** | Готовые уроки с фильтрами по предмету, возрасту учеников и продолжительности урока. Поиск и просмотр карточек. |
| 🔴 High | **Конструктор уроков** | Drag & drop редактор для создания уроков. Готовые блоки активностей. Экспорт в PDF для печати. |
| 🔴 High | **Управление заданиями** | Создание заданий, назначение классам/ученикам, установка дедлайнов. Автоматическая проверка тестов. |
| 🟡 Medium | **Класс-дашборд** | Список учеников в классе, их прогресс по заданиям, быстрый обзор активности. |
| 🟡 Medium | **Интеграция с Google Classroom** | Синхронизация классов, учеников и заданий с Google Classroom. |
| 🟡 Medium | **Интеграция с Microsoft Teams** | Создание заданий в Teams, синхронизация. |
| 🟢 Low | **Обмен материалами** | Публикация уроков для других учителей. Лайки, комментарии, рейтинг. |
| 🟢 Low | **Базовый аналитический отчёт** | Статистика посещаемости, процент выполненных заданий, средние оценки. |

### Подзаголовок
«Библиотека готовых уроков, конструктор материалов и управление классом. Экономьте до 5 часов в неделю и делитесь лучшими идеями с коллегами.»

### 3 пункта преимуществ
1. **⏱️ Экономия времени** — Готовые уроки и шаблоны, которые можно использовать сразу. Больше не нужно искать материалы по всему интернету.
2. **📚 5000+ материалов** — Уроки по всем предметам и возрастам. Фильтруйте по теме, классу и продолжительности.
3. **📊 Понятная аналитика** — Отслеживайте прогресс каждого ученика. Видьте, кто справляется, а кому нужна помощь.

### Секция «Как это работает» (3 шага)

**Шаг 1: Найдите урок**  
Откройте библиотеку и выберите готовый урок по вашему предмету и классу. Или создайте свой с нуля в конструкторе.

**Шаг 2: Настройте под себя**  
Отредактируйте материалы, добавьте свои задания и упражнения. Экспортируйте в PDF для печати или отправьте ученикам онлайн.

**Шаг 3: Отслеживайте результаты**  
Ученики выполняют задания, а вы видите их прогресс в реальном времени. Автоматическая проверка тестов экономит ваше время.

## 7. ТЕХНИЧЕСКИЕ РЕКОМЕНДАЦИИ

### Стек технологий

| Слой | Технология | Обоснование |
|------|------------|-------------|
| **Frontend** | React + Vite | Быстрая разработка, большое сообщество |
| **Роутинг** | React Router v6 | Стандарт для React SPA |
| **State** | Zustand | Лёгкий и простой state management |
| **Стили** | CSS Modules / Styled Components | Изоляция стилей |
| **Backend** | Node.js + Express | JavaScript fullstack, высокая производительность |
| **ORM** | Prisma | Типобезопасность, миграции |
| **База данных** | PostgreSQL | Надёжность, JSON поддержка |
| **Аутентификация** | Auth0 или Clerk | OAuth, магические ссылки, RBAC |
| **Файлы** | AWS S3 + CloudFront | Масштабируемость, CDN |
| **Хостинг** | Vercel (Frontend), Railway (Backend) | Лёгкий деплой, автомасштабирование |
| **PDF** | React-PDF или Puppeteer | Генерация PDF для печати |
| **Real-time** | Socket.io или Pusher | Уведомления, обновления |

### Интеграции

| Сервис | Назначение | API |
|--------|------------|-----|
| Google Classroom | Синхронизация классов и заданий | Google Classroom API |
| Microsoft Teams | Интеграция с Teams for Education | Microsoft Graph API |
| Zoom | Видеоуроки | Zoom SDK |
| Stripe | Платежи и подписки | Stripe API |
| Resend / SendGrid | Email-уведомления | REST API |
| OpenAI | AI-генерация (v2) | OpenAI API |

### Безопасность и compliance

**GDPR (ЕС):**
- Согласие на обработку данных
- Право на удаление данных
- Шифрование персональных данных
- DPA с облачными провайдерами

**FERPA (США):**
- Защита образовательных записей
- Контроль доступа родителей
- Логирование доступа к данным

**Общие меры:**
- HTTPS everywhere
- JWT с коротким TTL + refresh tokens
- Rate limiting
- Input validation (Zod)
- SQL injection protection (Prisma)
- XSS protection
- CORS configuration

## 9. KPI и МЕТРИКИ

### Активация
- **Sign-up to Class Setup Rate** — % пользователей, создавших класс после регистрации
- **Time to First Value** — время до создания первого урока/задания
- **Onboarding Completion Rate** — % завершивших onboarding

### Удержание
- **DAU/MAU** — соотношение дневных к месячным активным пользователям
- **Weekly Active Teachers** — активные учителя за неделю
- **Feature Adoption Rate** — % пользователей, использующих ключевые функции

### Монетизация
- **Trial to Paid Conversion** — конверсия из пробной в платную подписку
- **MRR** — ежемесячная повторяющаяся выручка
- **LTV** — пожизненная ценность пользователя
- **ARPU** — средняя выручка на пользователя

### Здоровье
- **Churn Rate** — % отписавшихся пользователей
- **NPS** — индекс лояльности
- **Support Tickets** — количество обращений в поддержку
